# AI Border Defense Surveillance System

An AI-powered intelligent border surveillance system designed to detect threats in real-time using computer vision and machine learning. The system analyzes surveillance video feeds, identifies suspicious activity, generates risk alerts, and visualizes high-risk zones using heatmaps.

This project demonstrates automated border monitoring using deep learning and real-time analytics.

---

## Project Overview

Traditional border surveillance requires continuous human monitoring, which is inefficient and prone to delays. This system uses AI-based object detection and motion analysis to automatically identify threats and assist security personnel.

The system processes video feeds and performs:

* Real-time object detection
* Suspicious movement detection
* Risk score calculation
* Threat alerts
* Predictive heatmap generation
* Intrusion zone monitoring
* Detection logging and backup
